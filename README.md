## Repository Structure

Solutions are organized by topic and problem.

```text

 ├── problem-name/
```

- revisit problems after learning new techniques.

## Problem Solving Approach

While solving problems, I try to:

- Understand the brute force solution first.
- Identify patterns and constraints.
- Optimize step by step.
- Analyze time and space complexity.
- Revisit difficult problems after some time.
